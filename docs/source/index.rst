Trainable Dokumentation
=======================
Trainable ist eine Open Source Webanwendung zur zielgerichteten
Trainingssteuerung.

Der Quellcode der Software sowie Möglichkeiten zur Meldung von Fehlern und
Verbesserungsvorschlägen finden sich bei `Trainable auf Github
<https://github.com/toirl/trainable>`_

.. important::
        Trainable ist derzeit noch in einem sehr frühen Entwicklungstadium und
        noch nicht bereit für den produktiven Einsatz. Diese Dokumentation
        dient zum aktuellen Zeitpunkt primär als Werkzeug zur Konzeptionierung
        der Anwendung. Viele der hier beschriebenen Funktionen sind noch nicht
        oder nicht vollständig implementiert.
